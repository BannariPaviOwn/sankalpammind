# Assets – Photo Placement Guide

Add your photos here and they will be used on the site. Use these exact filenames (or update `src/assets.ts` if you prefer different names). Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`.

## Folder structure

```
assets/
├── home/           → Home page
├── about/          → About page
├── initiatives/    → Initiatives page
└── get-involved/   → Get Involved page
```
